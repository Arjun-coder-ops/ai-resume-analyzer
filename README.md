## ✨ Introduction
AI Resume Analyzer is a modern web application that helps job seekers analyze and improve their resumes using Artificial Intelligence.  
You can upload your resume, receive personalized feedback, and instantly check how well your resume matches specific job descriptions.  
This project is designed to give ATS (Applicant Tracking System) insights, making resumes more competitive for job applications.  

---

## ⚙️ Tech Stack
- **React** → A flexible library for building user interfaces.  
- **React Router v7** → For navigation and routing between pages.  
- **Puter.js** → Adds authentication, storage, and AI capabilities directly in the browser.  
- **Tailwind CSS** → Utility-first framework for modern and responsive styling.  
- **TypeScript** → Provides type safety and improves code quality.  
- **Vite** → Fast build tool and development server.  
- **Zustand** → Simple and efficient state management.  

---

## 🔋 Features
- 🔐 **Authentication**: Secure browser-based login.  
- 📂 **Resume Upload & Storage**: Store and manage multiple resumes.  
- 🤖 **AI Resume Matching**: Get ATS scores, job matching percentages, and actionable feedback.  
- 🧩 **Reusable Components**: Modular UI for easy customization and extension.  
- 📱 **Responsive Design**: Works seamlessly across all devices.  
- 🎨 **Professional UI**: Built with Tailwind CSS for a sleek and modern design.  
- ⚡ **Performance Optimized**: Powered by Vite for fast builds and updates.

## 🤸 Getting Started

### Prerequisites
Make sure you have installed:
- [Git](https://git-scm.com/)  
- [Node.js](https://nodejs.org/) (npm is included with Node.js)  

### Installation
```bash
# Clone the repository
git clone https://github.com/goguarjun/ai-resume-analyzer.git
cd ai-resume-analyzer

# Install dependencies
npm install

# Start the development server
npm run dev
