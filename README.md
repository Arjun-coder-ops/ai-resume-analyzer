## ✨ Introduction

AI Resume Analyzer is a modern web app that helps you analyze and improve your resume using the power of AI. Upload your resume, get personalized feedback, and see how well your resume matches job listings—all in a clean, easy-to-use interface. This project is built for anyone looking to boost their job search with smart technology.

## ⚙️ Tech Stack

- **React**: A flexible JavaScript library for building user interfaces with reusable components and fast updates.
- **React Router v7**: Handles navigation and routing in the app, making it easy to manage different pages and views.
- **Puter.js**: A lightweight SDK that brings authentication, storage, and AI features right into your browser—no backend setup needed.
- **Tailwind CSS**: A utility-first CSS framework that lets you style your app quickly and responsively.
- **TypeScript**: Adds static typing to JavaScript, helping catch errors early and making the codebase more robust.
- **Vite**: A super-fast build tool and development server for modern web projects.
- **Zustand**: A minimal state management library for React, making global state easy and efficient.

## 🔋 Features

- **Simple Authentication**: Sign in securely right from your browser—no complicated setup required.
- **Resume Upload & Storage**: Keep all your resumes organized and safe in one place.
- **AI-Powered Resume Matching**: Instantly see how your resume stacks up against job descriptions, with tailored feedback and ATS scores.
- **Reusable UI Components**: Clean, modular components make the app easy to maintain and extend.
- **Responsive Design**: Works smoothly on any device, from desktop to mobile.
- **Modern Look & Feel**: Built with Tailwind CSS for a sleek, professional appearance.
- **Efficient Codebase**: Modular structure and reusable code for easy updates and new features.

## 🤸 Getting Started

Follow these beginner-friendly steps to set up and run the project on your computer.

**What You Need First**

- [Git](https://git-scm.com/) (for downloading the project)
- [Node.js](https://nodejs.org/en) (to run JavaScript code)
- [npm](https://www.npmjs.com/) (comes with Node.js, used to install packages)

**How to Download the Project**

Open your terminal or command prompt and run:

```bash
git clone https://github.com/goguarjun/ai-resume-analyzer.git
cd ai-resume-analyzer
```

**How to Install Everything**

Install all the necessary packages by running:

```bash
npm install
```

**How to Start the App**

Start the development server with:

```bash
npm run dev
```

Then, open [http://localhost:5173](http://localhost:5173) in your web browser to see the app in action.
